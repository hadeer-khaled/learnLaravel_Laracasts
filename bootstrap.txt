1. download bootstrap => https://getbootstrap.com/docs/5.3/getting-started/download/

2. in the public folder in your Laravel project
	- create css folder and put these files inside it
		- bootstrap.css
	        - bootstrap.min.css

	- create js folder and put these files inside it
		- bootstrap.js
	        - bootstrap.min.js

3. in your view

        -     <link rel="stylesheet" href="{{assets('css/bootstrap.css')}}">
        -     <link rel="stylesheet" href="{{assets('css/bootstrap.min.css')}}">
	-     <script src="{{assets('js/bootstrap.js')}}"></script>
    	-     <script src="{{assets('js/bootstrap.min.js')}}"></script>
